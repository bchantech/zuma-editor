Key Cheat Sheet
  • A Append point at end
  • S Split line segment
  • D Delete point
  • O / P Lower/Raise Z-index
  • T Toggle tunnel
  • W Toggle warp
  • R Reset and center the view
  • B Toggle background
  • Z Undo
  • Shift+R Reflect curve horizontal
  • Shift+V Reflect curve vertical
  • Right Click Pan
  • Mouse Wheel Zoom

Zuma Deluxe Map restrictions:

  • Only first curve will be saved
  • Warp nodes are not supported
  • Map metadata is not saved with the map, define them in levels.xml

OpenSMCE Map restrictions:

  • Map name will need to be entered in config.json after saving
  • tunnels.json will need to be defined if your map has tunnels
  • Shooter location will have to be manually defined
  • Map metadata is not saved with the map, define them in the level .json
